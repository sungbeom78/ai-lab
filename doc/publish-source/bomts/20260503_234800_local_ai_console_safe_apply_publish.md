# AI Hub Safe Apply 릴리스 노트

**버전/일자**: 2026-05-03

이번 릴리스에서는 Local AI Console에 **Safe Apply** 기능이 추가되었습니다. 이 기능은 Qwen 모델 등에서 제안된 파일 시스템 변경 사항을 시스템에 즉시 반영하지 않고, 검증된 안전한 "작업 템플릿"을 통해서만 승인된 파일 내역을 생성 및 덮어쓰기 하는 구조입니다.

### 주요 변경 사항
* **보안성 강화된 API**: `preview` 기능 제공으로 덮어쓰기 여부 및 민감 정보 차단 내역을 미리 확인할 수 있습니다.
* **민감 정보 탐지**: 파일 내에 `token`, `secret`, `password` 등이나 외부 `http://` 링크가 포함되면 쓰기가 자동차단됩니다.
* **백업 지원**: 기존 파일 존재 시 덮어쓰기 전 `.bak_날짜_시간` 형식으로 백업을 수행합니다.
* **새로운 UI**: Local AI Console 화면에서 직접 ahnda MVP를 미리보고 적용할 수 있는 버튼이 신설되었습니다.

안전한 AI 운영 워크플로우를 위해 설계되었으며, 추후 다양한 "Safe Apply" 템플릿 확장으로 이어질 예정입니다.
