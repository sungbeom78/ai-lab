# 작업 로그: 11004 포트 이사와 VENV 안정화

- Date: 2026-05-03

결국 1004번 포트는 Linux에서 관리자 권한(root)이 필요한 포트라 매번 sudo를 입력해야 하는 번거로움이 있었다.
그래서 11004 포트로 깔끔하게 이사했다!

더불어 python 패키지가 꼬이는 것을 막기 위해 `.venv` 환경 기반으로 격리해서 실행되도록 스크립트를 재정비했다.
이로써 노트북에서도 이 데스크탑의 AI 환경을 웹 콘솔로 당겨 쓸 수 있는 밑작업이 끝났다.
방화벽과 포트포워딩(portproxy) 승인만 떨어지면, 밖에서도 내 마음대로 Ollama 로컬 모델을 돌릴 수 있다!
