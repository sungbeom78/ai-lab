# Ahnda 작업 로그: Safe Apply 기능 도입 준비

**일시**: 2026-05-03 23:48
**작업자**: Desktop AI
**관련 프로젝트**: ahnda, ai-hub

오늘은 Local AI Console에서 생성된 파일들을 로컬 파일 시스템에 안전하게 적용하기 위한 "Safe Apply" 기능을 구현했습니다.

Qwen 모델의 출력을 그대로 신뢰하는 대신, 미리 정의된 안전한 MVP 형태의 파일만을 `/project/site/ahnda` 하위에 생성할 수 있도록 백엔드 라우터와 프론트엔드 버튼을 구성했습니다. 이 과정에서 민감 정보(비밀번호, 토큰 등) 검출 로직과 외부 CDN 호출 방지 로직도 추가하여 보안성을 높였습니다. 

현재는 API와 UI 구현이 완료된 상태이며, 최종적으로 사용자의 승인이 떨어지면 적용 버튼을 통해 Ahnda 개인 작업실의 첫 MVP(다크톤 카드형 UI)가 브라우저에 나타나게 될 것입니다. 이를 통해 개발 속도를 안전하고 빠르게 가져갈 수 있는 파이프라인이 완성되어 가고 있습니다.
