## 작업 내용 (Goal)
1. 데스크탑 AI(WSL) 환경에 Ollama와 통신하는 OpenClaw Bridge Server(FastAPI) 구축
2. Windows 환경에서 사용할 수 있는 OpenClaw VS Code Extension(TypeScript) 뼈대 코드 생성

## 실행 내역 (Actions)
- 기존 AI Hub(port 11004)와 충돌을 피하기 위해 OpenClaw Bridge Server 포트를 `11006`으로 지정.
- `router.py`를 통해 사용자 입력 키워드("코드", "작성해" 등)에 따라 `gemma3:4b`와 `qwen2.5-coder:7b`를 자동 라우팅하도록 구현.
- `session_logger.py`를 추가하여 대화 이력과 컨텍스트(작업 폴더, 현재 파일 등)를 로컬 마크다운 파일로 저장하도록 구성.
- 백그라운드로 `11006` 포트 서버를 띄운 후 `/api/health` 및 `/api/chat` 통신 테스트 완료. (코드 라우팅 동작 시 `qwen` 7B 모델 로드로 인해 지연되는 현상 확인 및 중지 조치)
- VS Code 확장을 위한 폴더 `/project/openclaw-vscode-extension`를 생성하고 `package.json`, `tsconfig.json`, `extension.ts`, `api.ts`, `OpenClawViewProvider.ts` 코드 작성 완료.
